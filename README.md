## üé® UI/UX Design Philosophy & Innovation
I defined the visual identity of the platform, balancing **academic rigor** with **child-friendly engagement**.

### üåø Jungle-Themed Engagement
- **Thematic Consistency:** Designed a vibrant *Jungle* theme to reduce child anxiety during assessments and improve task persistence.
- **UI Fundamentals:** Applied core UI principles including **Visual Hierarchy** (directing attention to the marble), **Consistency** (unified button styles), and **Accessibility** (high-contrast elements for young users).
- **Interactive Feedback:** Designed and implemented a **Randomized Firework Celebration System** on the results page. Using Phaser‚Äôs particle system, multi-colored fireworks are triggered as positive reinforcement to reward task completion.
